Output for httpbin

  • Httpbin tests the form's input information and delivers the output in a simple ordered list. It puts everything into perspective, both form and server information. Though both are displayed httpbin output reflects what information was entered in the form at the top. The output feedback gives a visual of the information which can be reviewed and then revised in the code.
